Campaign Manager Console

CPaaS Campaign Manager Console allows Communications Providers to create, manage, or delete enterprises (business customers) through their own Restcomm account.

This document contains information for Communications Providers regarding the Campaign Manager Console.

Console Overview

You can access the Campaign Manager Console by navigating to https://yourdomain.com/campaignmanagerconsole/.

The users drop-down at the top right corner of the console contains the following:

  • Account: Click to view your account details

  • Volume: Click to view volume transactions per Enterprises

  • Sign Out: Click to sign out of the application.

Next to it is a link to visit Documentation.

The Campaign Manager console contains two tabs:

Enterprise: Here you can create, manage, and delete enterprises.
Campaigns: Here you can view campaigns created under all the enterprises that were created by this account.

Volume

This feature allows you to assign and manage volumes to Enterprises.

  • Assign volume: You can assign volume when you create an enterprise. Enterprises require volume to launch campaigns. The volume is consumed based on the number of campaign recipients and the campaign message length. The unused volumes from a campaign are restored to the Enterprises.

  • Manage volume: You can add or deduct volumes from enterprises thereby having complete control over Enterprises. This also helps in fraud prevention. For example, if an enterprise is compromised, they wont be able to exceed their Volume quota

The Volume button displays the volume transactions of all enterprises.

Volume transactions can be filtered based on the following:

  • Start Date: Displays all Volume transactions that took place on or after this date.

  • End Date: Displays all Volume transactions that took place on or before this date.

  • Transaction Sid: Displays the Volume transaction that is associated with the entered transaction Sid value.

  • Type: Displays all Volume transactions based on the type of transaction. The following options are available:

    • All Types: Displays all volume transactions

    • Reduction: Displays transactions involving volume reductions.

    • Addition: Displays transactions involving volume additions

    • Campaign: Displays transactions of volumes consumed by campaigns

    • Restore: Displays transactions involving volume restorals. This could be because of unused volumes or a canceled campaign.

  • Enterprise Name: Displays Volume transactions of enterprises that match the name entered

Enterprises

The Enterprises tab allows you to create, manage, and delete enterprises.

Users can view a list of all enterprises that have been added under this account. Enterprises can be filtered based on the following:

  • Start Date: Displays all the enterprises created on or after this date.

  • End Date: Displays all the enterprises created on or before this date.

  • Name: Displays all enterprises that matches the name entered partially or fully.

  • Status: Displays all enterprises that match the selected status. The following status is available to select:
    All Status
    Active
    Inactive Closed

Define the above filter fields and click Search to view the desired enterprises. Click Clear all if you want to clear the filter fields.

The Enterprise page provides the option of displaying 10, 20, or 50 items in a single page. You can select this number from the Items per page drop-down list located at bottom of the page.

The following page navigation options can be used while viewing the enterprise list:

  • First Page

  • Previous Page

  • Next Page

  • Last Page

Adding an Enterprise

To add an enterprise, do the following:

  • Click on the Add Enterprise button in the Enterprise page. Add Enterprise window appears as shown below:

  • Enter the name of the new enterprise.

  • Enter an administrator name for the enterprise. .

  • Enter the administrator email address. This email must be used to login to this enterprise once created.

  • Enter the password for the administrator. This will be set in the Restcomm core platform. This password must be used to log in to this enterprise once created.

  • Enter the throttle value as explained below

    • smsPerSecond: represents maximal SMS/sec rate allowed to this Enterprise.

    • callsPerSecond: represents maximal calls/sec rate allowed to this Enterprise.

    • concurrentCalls: represents the number of concurrent calls allowed to this Enterprise at any given point in time.

  • Add the amount of volume for the Enterprise.

  • Click Save. The newly created enterprise will be displayed in the Enterprise page.

Whenever a Communications Provider adds a new Sub Account for a Business Customer, the system creates its user record under the Restcomm Core platform as a Sub Account of the Communications Provider Account.
The Enterprise name and Administrator Email address must be unique.

Passwords should meet the following criteria:
- At least one Uppercase letter (A,B,C,D)
- At least one Lowercase letter (a,b,c,d,.)
- At least one number (1,2,3,4,5.)
- At least one special character (!,@,#,$,%,&,.)
- Length of the password should be greater than or equal to 8.

Editing an Enterprise

  • To edit the details of an enterprise, click on the enterprise row. The enterprise details appear in a pop-up window. The following parameters can be changed:
    Name of the enterprise
    Throttle values
    Status: You can activate or inactivate an enterprise.

  • Click Save after making the required changes.

Deleting an Enterprise

To delete an enterprise, click on the enterprise. The enterprise details appear in a pop-up window. Click on the Delete button on the bottom of the window. This action cannot be undone. Once the account is deleted you will not be able to restore it.

When an enterprise is deleted, the Admin user of that enterprise are CLOSED in CPaaS.

Manage Volume

You can manage Volume of Active Enterprises. To add or reduce volume for an enterprise, do the following:

  • Click on Manage Volume icon against the enterprise. Manage Volume window appears.

  • Select the value Add or Reduce from the Modify Volume drop-down list.

  • Enter a description for the addition or reduction of volume.

  • Click Confirm. You can click Cancel to exit this window.

Campaigns

The Campaigns page lists all the campaigns created by enterprises under this account.

You can search campaigns based on the following filter parameters:

  • Start Date: Displays all the campaigns that were created on or after this date.

  • End Date: Displays all the campaigns that were created on or before this date.

  • Campaign name: Displays all campaigns matching with the name entered, partially or fully.

  • Status: Displays all campaigns that match the selected status. The following statuses are available to select:
    Initiating: System is processing the Campaign for launch
    Scheduled: Campaign has initiated and now pending its schedule to launch
    In Progress: Campaign is in progress, sending SMSs or calling the To Numbers
    Completed: Campaign has completed successfully
    Failed
    Canceled: Campaign disrupted due to internal system error or if Campaign was On Progress while its parent Enterprise was deleted.

  • Channel: Displays all campaigns that match the selected channel. You can select:
    All Channels
    SMS
    Voice

  • Enterprise Name: Displays all the campaigns created under the selected enterprise.

  • Enterprise Sid: Displays all the campaigns under the Enterprise associated with the entered sid.
