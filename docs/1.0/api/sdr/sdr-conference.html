Conference SDR

Conference records are generated when applications execute the <Dial Conference> verb.

Format

A Conference SDR record comprises the following fields.

Field Description Type

sid

The record identifier.

string

org_sid

The organization to which the record belongs to.

string

org_name

The domain name of the Organization which belongs to the Account.

string

account_sid

The account to which the record belongs to.

string

account_friendly_name

The friendly name of the Account.

string

api_version

The API version that generated the record.

string

date_created

The date when the record was created.

timestamp (format: Thu, 01 Jan 2022 00:20:00 +0000)

date_created_ms

The datetime in milliseconds when the record was created.

timestamp (format: Thu, 01 Jan 2022 00:20:00.123 +0000)

date_updated

The date when the record was last modified.

timestamp (format: Thu, 01 Jan 2022 00:20:00 +0000)

date_updated_ms

The datetime in milliseconds when the record was last modified.

timestamp (format: Thu, 01 Jan 2022 00:20:00.123 +0000)

friendly_name

The name of the conference room.

completed, failed

status

The status of the conference call.

completed, failed

uri

The URI that references the SDR record.

URI

subresource_uris.participants

The URI that references the list of participants in the call.

URI
