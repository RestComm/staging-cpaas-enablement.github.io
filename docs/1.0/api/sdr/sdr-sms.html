SMS SDR

Sms records are generated when applications execute the <SMS> verb.

Format

An SMS SDR record comprises the following fields.

Field Description Type

sid

The record identifier.

string

account_sid

The account to which the record belongs to.

string

account_friendly_name

The friendly name of the Account.

string

org_sid

The organization to which the record belongs to.

string

org_name

The domain name of the Organization which belongs to the Account.

string

date_created

The date when the record was created.

timestamp (format: `Thu, 01 Jan 2022 00:20:00 +0000)

date_created_ms

The datetime in milliseconds when the record was created.

timestamp (format: Thu, 01 Jan 2022 00:20:00.123 +0000)

date_updated

The date when the record was last modified.

timestamp (format: Thu, 01 Jan 2022 00:20:00 +0000)

date_updated_ms

The datetime in milliseconds when the record was last modified.

timestamp (format: Thu, 01 Jan 2022 00:20:00.123 +0000)

from

The sender ID.

string

to

The recipient ID.

string

body

The message body.

string

num_segments

If the message is sent to a PSTN and requires segmentation because of length, this will indicate the number of segments.

integer

sip_status_code

The SIP error code response to the SIP message request.

string

sip_status_reason

The SIP response error reason to SIP message request.

string

status

The SMS message status: received, sent, delivered, undelivered, failed.

string

direction

The message direction: inbound, outbound-call, outbound-api.

string

price

The price of the record.

double

price_unit

The unit of the price.

string

api_version

The API version that generated the record.

string

uri

The URI that references the SDR record.

URI

error_code

The internal error code.

integer

error_message

The internal error message.

string

reason.message

The message explaining the session failure.

string

reason.code

The reason code explaining the session failure. Relates to reason.protocol.

integer

reason.protocol

The protocol that originated the session failure. Relates to reason.code.

string

media_list.item_*.url

The URL of the media attachment for this SMS message.

URI

media_list.item_*.content_type

Content Type of the media attachment for this SMS message.

string

media_list_size

Number of media attachments.

integer

headers.*

Any custom header associated with this SMS message. May result in multiple columns or none.

string
