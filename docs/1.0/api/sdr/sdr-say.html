Play SDR

Say records are generated when applications execute the <Say> verb.

Format

A Say SDR record comprises the following fields.

Field Description Type

sid

The record identifier

string

org_sid

The organization to which the record belongs to

string

org_name

The domain name of the Organization which belongs to the Account.

string

account_sid

The account to which the record belong to

string

account_friendly_name

The friendly name of the Account.

string

call_sid

The voice session to which the record belongs to

string

api_version

The API version that generated the record

string

date_created

The date when the record was created

timestamp (format: Thu, 01 Jan 2022 00:20:00 +0000)

date_created_ms

The datetime in milliseconds when the record was created

timestamp (format: Thu, 01 Jan 2022 00:20:00.123 +0000)

provider

The chosen provider to synthesize the message

string

voice

The chosen voice to synthesize the message

string

language

The language code of the message

string

loop

How many times the media was set to be played

integer

duration

How long the media was played, in seconds

integer

character_count

Character count of the synthesized message

integer

status

The execution status of the Play operation

completed, failed

reason

The reason why the operation failed

string

uri

The URI that references the SDR record

URI
