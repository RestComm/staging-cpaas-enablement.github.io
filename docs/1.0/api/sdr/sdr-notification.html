Notification SDR

Notification records are generated for any issue occurred in an incoming or outgoing traffic.

Format

A Notification SDR record comprises the following fields.

Field Description Type

api_version

The API version that generated the record.

string

sid

A string that uniquely identifies this notification.

string

date_created

The date on which this notification was created.

timestamp (format: Mon, 04 Jul 2022 01:46:26 +0000)

date_created_ms

The datetime in milliseconds on which this notification was created.

timestamp (format: Mon, 04 Jul 2022 01:46:26.123 +0000)

date_updated

The date on which this notification was last updated.

timestamp (format: Thu, 01 Jan 2022 00:20:00 +0000)

date_updated_ms

The datetime in milliseconds on which this notification was last updated.

timestamp (format: Thu, 01 Jan 2022 00:20:00.123 +0000)

account_sid

The unique id of the Account that created this notification.

string

call_sid

The unique id of the call which generated this notification.

string

log

An integer log level corresponding to the type of notification. 0 for an ERROR, and 1 for a WARNING.

int

error_code

A unique error code for the error condition.

string

more_info

More information about this notification.

string

message_text

The text of the notification.

string

message_date

The date on which this notification message was generated.

timestamp (format: Thu, 01 Jan 2022 00:20:00 +0000)

message_date_ms

The datetime milliseconds on which this notification message was generated.

timestamp (format: Thu, 01 Jan 2022 00:20:00.123 +0000)

request_url

The Url of the RCML/Status callback which is causing an issue.

string

request_method

HTTP REST method was used by request_url

string

request_variables

Data was sent to request_url

string

response_headers

Headers from the response generated by the request_url server.

string

response_body

Body from the response generated by the request_url server.

string

org_sid

The unique id of the Organization which belongs to the Account.

string

uri

The URI that references the SDR record.

URI
