Campaign Manager Overview

Introduction

CPaaS account holders can increase engagement with the right SMS/Voice campaign. Using CPaaS Campaign Manager (CM) you can easily launch an SMS or voice campaign to hundreds of thousands of phone numbers. You can use the APIs or intuitive console to create and manage campaigns. The console provides an easy way to create multiple announcements, number lists, and DND lists to use in creating targeted campaigns.

Features and Benefits

According to Gartner, viewing SMS messages are as high as 98 percent compared with only 20 percent for email messages. Organizations increasingly choose SMS to communicate important content. CPaaS Campaign Manager offers competitive features including:

  • Intuitive Console: Manage different capabilities of Campaign Manager like Enterprises, usage records, and more with an intuitive console.

  • Scale: Single Campaign can contain up to 1 million targets.

  • Volume: Easily plan for scale of campaigns based on the available volume.

  • Voice Announcements: Create Voice announcements using custom audio file or Text-to-Speech service.

  • Content Personalization: User can easily define message templates which allows adjusting the message per each individual target phone number.

  • Batch File Upload Support: The content of the uploaded files is verified to minimize the operational costs during the file upload process.

  • Do not Disturb (DND): Set Opt-out and Opt-In list of phone numbers.

  • Campaign Scheduling: Scheduled and immediate Campaigns execution.
