Click-to-Call Overview

Overview

A Click-to-Call feature makes it possible to add a button to your applications to make WebRTC calls.

With WebRTC Click-to-Call capability, you can now enable your customers to call a pre-configured destination by clicking a button in their application. When a user clicks on that button, a WebRTC call is established from the application itself to the given destination, and the user can talk to the destination using their device.

The Click-to-Call capability can be accomplished easily by adding the CPaaS SDKs for Web, iOS, and Android to your application.

The WebRTC Click-to-Call function adds WebRTC capabilities to existing Programmable Voice services, allowing you to further enhance your Programmable Voice experience.

WebRTC

WebRTC (Web Real-Time Communication) is a collection of communications protocols and application programming interfaces that enables real-time communication over peer-to-peer connections or more.

Network Traversal Service

Our solution includes a reliable TURN server to relay the media around the corporate firewall and transport data over even the most stringent networks.
